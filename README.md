# grails-api-echo

Description:
--------------
Grails plugin, used to help debug REST APIs.

Installation:
--------------
```
  plugins {
...
    compile ':api-echo:0.2'
...
  }
```

Usage:
--------------
{{HTTP_METHOD}} {{YOUR_URL}}/echo/{{API_TO_ECHO}}
